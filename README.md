# Student Management System (Python)

This is a simple Student Management System built using Python.
It allows users to add, view, and search student records.
Student data is stored using file handling.

## Features
- Add student details
- View all students
- Search student by roll number
- Grade calculation

## Technologies Used
- Python
- File Handling

## How to Run
1. Run main.py
2. Follow on-screen menu
